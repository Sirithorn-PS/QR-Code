  - ปรับปรุงกล่องป๊อปอัป "เพิ่มรายการสินค้า" ในหน้าจัดการสต็อก (`app/inventory/page.tsx`) ให้มีมาตรฐานการแสดงผลบนมือถือรูปแบบเดียวกัน โดยยกกล่องลอยเหนือแถบเมนูด้านล่าง 72 พิกเซล จำกัดความสูงพร้อมระบบเลื่อนสกอร์ภายในกล่อง และปรับช่องกรอกจำนวนเริ่มต้นให้พิมพ์เลขได้อย่างอิสระโดยไม่มีปุ่มลูกศรรบกวน เพื่อให้ใช้งานบนหน้าจอมือถือได้อย่างสะดวกไม่โดนบัง
  - แก้ไขปัญหาข้อความแจ้งเตือนเมื่อเพิ่มสินค้ารหัสซ้ำ (`Product with this Item Code already exists`) โดยแปลข้อความข้อผิดพลาดจาก Backend API (`backend/src/index.ts`) และ Frontend (`app/inventory/page.tsx`) ให้เป็นภาษาไทยที่อ่านเข้าใจง่าย เช่น "รหัสสินค้า (Item Code) นี้มีอยู่ในระบบแล้ว กรุณาใช้รหัสอื่น" พร้อมย้ายตำแหน่งแสดงแถบแจ้งเตือนสีแดงมาไว้ภายในกล่องป๊อปอัปทันที ทำให้ผู้ใช้งานรู้ตัวและแก้ไขรหัสได้ต่อเนื่องโดยไม่ต้องปิดหน้าต่างป๊อปอัป
